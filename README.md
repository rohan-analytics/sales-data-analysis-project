# sales-data-analysis-project

Sales Data Analysis Project
This is an end-to-end sales data analysis project created using SQL, Excel, and Power BI.
The goal of this project is to analyze sales performance, profit, category trends, and state-wise performance to generate useful business insights.


Tools Used:
Microsoft Excel (Data Cleaning)
MySQL (Data Analysis & Queries)
Power BI (Dashboard & Visualization)


Project Overview
In this project, I cleaned and analyzed raw sales data and built an interactive Power BI dashboard.

The dashboard includes:
Total Sales, Profit, Orders and Quantity KPIs
Monthly sales trend analysis
Category-wise and state-wise performance
Target vs Actual sales comparison
Interactive filters for better analysis
This project demonstrates my ability to clean data, analyze performance, and create business insights using SQL and Power BI.


Project Files Included:

Dataset (CSV files)
SQL queries file
Power BI dashboard (.pbix)
Project report (PDF)
Dashboard screenshot


Key Insights :
Sales are high but profit margin is relatively low
Electronics category drives highest sales
Clothing category generates highest profit
Sales peak at start and end of year
Some states contribute more to overall revenue
Sales targets are not fully achieved
